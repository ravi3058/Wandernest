Deployment Guide

This is a Next.js project that can be deployed using Vercel or any other Next.js-compatible hosting service.

  1. Clone the repository: git clone https://github.com/yourusername/wandernest.git
  2. Install dependencies: npm install or yarn install
  3. Run development server: npm run dev or yarn dev
  4. Build for production: npm run build or yarn build
  5. Deploy to Vercel: Connect your GitHub repository to Vercel for automatic deployments
